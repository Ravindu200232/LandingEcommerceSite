# 🛍️ LandingEcommerceSite

A modern, responsive, and visually appealing **Landing Page for an E-commerce Store** built using **Next.js**, **React**, and **shadcn/ui**. Designed with performance, accessibility, and user experience in mind.

🔗 **Live Demo**: [landing-ecommerce-site.vercel.app](https://landing-ecommerce-site.vercel.app/)

---

## ✨ Features

- ⚡ Built with **Next.js 15.4**
- 💅 Styled using **shadcn/ui + Tailwind CSS**
- 🎯 Fully responsive and mobile-friendly
- 🔥 Eye-catching hero section with call-to-action
- 🧩 Modular and reusable React components
- 🌐 SEO-friendly with optimized metadata
- 🚀 Deployed on **Vercel**
